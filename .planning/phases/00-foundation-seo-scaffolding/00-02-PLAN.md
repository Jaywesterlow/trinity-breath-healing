<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| route load() → <Head> prop | Typed PageMeta only; no path-derived or query-derived strings injected raw |
| <svelte:head> emission → prerendered HTML | Bytes go to public crawlers — must not contain injected scripts or unescaped user content |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-00-meta | Tampering | <Head> emission via <svelte:head> | mitigate | All meta values come from typed `PageMeta` populated by `+page.ts` load() — never from `url.searchParams`, never from path-pattern interpolation of user input. Svelte's built-in attribute escaping handles any embedded quotes. Plan 05 enforces the same discipline on stub pages. |
| T-00-noindex-preview | Information Disclosure | Vercel preview URLs indexed by search engines | accept (deferred to Plan 08) | The `meta.noindex` plumbing lands here; Plan 08 wires the env-conditional check (`PUBLIC_VERCEL_ENV !== 'production'` → noindex=true). Plan 02 only ships the plumbing. |

</threat_model>
