## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Manual `paths` in tsconfig.json | Adding `$schema/*` / `$content/*` paths in tsconfig.json overrides SvelteKit's auto-generated `$lib` alias → svelte-check error. Kit.alias is single source of truth. | advisory | Never add path aliases to tsconfig.json — use `kit.alias` in svelte.config.js only. |
| jsdom missing in Vitest 4 | Vitest 4 dropped bundled jsdom; unit environment fails with `Cannot find package 'jsdom'` | advisory | Any new project using `environment: 'jsdom'` must add `jsdom` to devDependencies explicitly. |
